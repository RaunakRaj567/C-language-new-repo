# C Language Learning Portal 🚀

A comprehensive, interactive web-based learning platform for mastering C programming from zero to hero! This portal includes detailed lessons, practical examples, code snippets, and viva questions for all 7 units of C language basics.

## 🎯 7 Complete Units

### 📚 Unit 1: C Language Basics
Character set, Identifiers, Keywords, Data types, Variables, Constants, Input/Output

### ⚡ Unit 2: Control Structures
if/else, switch, while, for, do-while, break, continue statements

### 🔧 Unit 3: Functions & Storage Classes
Function definition, recursion, scope rules, storage classes (static, extern, register)

### 📦 Unit 4: Arrays in C
1D & 2D arrays, strings, linear/binary search, bubble/selection sort
**Status: ✅ Fully English**

### 🎯 Unit 5: Pointers & Strings
Pointers, memory management (malloc, calloc, free), string functions
**Status: ✅ Fully English**

### 🏗️ Unit 6: Structures & Unions
Structure declaration, nested structures, unions, self-referential structures

### ⭐ Unit 7: C++ Basics
Classes, objects, inheritance, polymorphism, exceptions, templates, STL

## 🚀 Quick Start

### Prerequisites
- Node.js v14+
- npm

### Installation

```bash
# Clone repository
git clone https://github.com/RaunakRaj567/C-language-new-repo.git
cd C-language-new-repo

# Install dependencies
npm install

# Start development server
npm start
```

The application opens at `http://localhost:3000`

### Production Build

```bash
npm run build
```
